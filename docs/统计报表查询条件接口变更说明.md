# 统计报表 — 查询条件接口变更说明

> 日期：2026-06-10

4 个报表接口新增可选查询参数，不传参时行为与原有接口完全一致，前端按需对接。

---

## ① 报名趋势 — 支持自定义日期范围

### 接口

```http
GET /statistics/registration-trend
```

### 新增参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| startDate | String | 否 | 开始日期，格式 `yyyy-MM-dd`，默认30天前 |
| endDate | String | 否 | 结束日期，格式 `yyyy-MM-dd`，默认今天 |

### 示例

```http
GET /statistics/registration-trend?startDate=2026-01-01&endDate=2026-06-10
GET /statistics/registration-trend                      ← 默认近30天
```

### 响应

与原有格式一致（ECharts line），`title.text` 固定为 `"报名趋势"`。

---

## ② 通过率趋势 — 支持按年份和科目筛选

### 接口

```http
GET /statistics/pass-rate
```

### 新增参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| year | int | 否 | 年份，如 `2026`，不传查全部 |
| subject | int | 否 | 科目：1-4，不传查全部科目 |

### 示例

```http
GET /statistics/pass-rate?year=2026&subject=1     ← 只看2026年科目一
GET /statistics/pass-rate?year=2026                ← 只看2026年全部科目
GET /statistics/pass-rate                          ← 查全部历史
```

### 响应

与原有格式一致（ECharts 多 line）。

---

## ③ 教练效能排名 — 支持按车型和 TopN 筛选

### 接口

```http
GET /statistics/coach-workload
```

### 新增参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| licenseType | String | 否 | 准教车型，如 `C1`，不传查全部教练 |
| topN | int | 否 | 返回前 N 名，如 `5`，不传返回全部 |

### 示例

```http
GET /statistics/coach-workload?licenseType=C1&topN=5   ← C1教练排名前5
GET /statistics/coach-workload?topN=10                  ← 全部教练排名前10
GET /statistics/coach-workload                          ← 查全部
```

### 响应

与原有格式一致（ECharts bar + detailData）。

---

## ④ 收入看板 — 支持按年份筛选

### 接口

```http
GET /statistics/revenue-summary
```

### 新增参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| year | int | 否 | 年份，如 `2026`，不传时近12月趋势 + 当月来源 |

### 示例

```http
GET /statistics/revenue-summary?year=2026     ← 查看2026全年
GET /statistics/revenue-summary               ← 默认近12月+当月
```

### 响应变化

传 `year` 时，饼图标题从 `"当月收入来源"` 变为 `"2026年收入来源"`，月度趋势标题从 `"近12个月收入趋势"` 变为 `"2026年收入趋势"`。响应结构不变。

---

## 汇总对照

| 报表 | 新增参数 | 默认行为 |
|------|----------|----------|
| ① 报名趋势 | `startDate`, `endDate` | 近30天 |
| ② 通过率趋势 | `year`, `subject` | 全部历史 |
| ③ 教练效能排名 | `licenseType`, `topN` | 全部教练 |
| ④ 收入看板 | `year` | 近12月+当月 |

所有参数均为可选，不传时行为与改动前完全一致。
