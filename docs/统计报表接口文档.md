# 统计报表 — 前端接口文档

## 通用说明

### 统一响应格式

```json
{
  "state": 20000,
  "message": null,
  "data": { ... }
}
```

| state | 含义 |
|-------|------|
| 20000 | 成功 |
| 40000 | 请求参数错误 |
| 40100 | 未登录或 token 过期 |
| 40300 | 权限不足 |
| 40400 | 资源不存在 |
| 40900 | 数据冲突 |

### 认证方式

```
Authorization: Bearer <token>
```

### 角色说明

| 值 | 角色 |
|----|------|
| 1 | 学员 |
| 2 | 教练 |
| 3 | 管理员 |

> **所有统计报表接口仅管理员可调用。**

基础路径：`/statistics`

---

### 1. 报名趋势（折线图）

```
GET /statistics/registration-trend
```

> 近30天每日报名人数，返回 ECharts line 格式。

**响应 data：**
```json
{
  "title": { "text": "近30天报名趋势" },
  "xAxis": { "type": "category", "data": ["2026-05-06", "2026-05-07", "..."] },
  "yAxis": { "type": "value", "minInterval": 1 },
  "series": [
    { "name": "报名人数", "type": "line", "smooth": true, "data": [0, 1, 0, 3, ...] }
  ]
}
```

**前端渲染：** 直接传入 ECharts option。

### 2. 各科目月度通过率趋势（折线图）

```
GET /statistics/pass-rate
```

> 按科目拆分、按月展示考试通过率变化趋势，返回 ECharts 多 line 格式。

**响应 data：**
```json
{
  "title": { "text": "各科目月度考试通过率趋势" },
  "tooltip": { "trigger": "axis" },
  "legend": { "data": ["科目一", "科目二", "科目三", "科目四"] },
  "xAxis": { "type": "category", "data": ["2026-01", "2026-02", "2026-03", "..."] },
  "yAxis": { "type": "value", "min": 0, "max": 100, "name": "通过率(%)" },
  "series": [
    { "name": "科目一", "type": "line", "connectNulls": false, "data": [95.0, 92.5, null, ...] },
    { "name": "科目二", "type": "line", "connectNulls": false, "data": [80.0, 75.0, 85.0, ...] },
    { "name": "科目三", "type": "line", "connectNulls": false, "data": [88.0, null, 90.0, ...] },
    { "name": "科目四", "type": "line", "connectNulls": false, "data": [null, 100.0, 95.0, ...] }
  ]
}
```

**说明：**
- 各科目分别一条折线，无数据的月份为 `null`（不连线）
- Y 轴范围 0-100，单位 %

### 3. 教练效能排名（柱状图）

```
GET /statistics/coach-workload
```

> 按学员考试通过率排名，附带评分/带教学员数/执教年限明细。

**响应 data：**
```json
{
  "title": { "text": "教练效能排名" },
  "tooltip": { "trigger": "axis" },
  "xAxis": { "type": "category", "data": ["张教练", "李教练"] },
  "yAxis": { "type": "value", "min": 0, "max": 100, "name": "通过率(%)" },
  "series": [
    { "name": "考试通过率(%)", "type": "bar", "data": [90.0, 75.0] }
  ],
  "detailData": [
    {
      "coachName": "张教练",
      "rating": 4.8,
      "coachYears": 6,
      "studentCount": 15,
      "examCount": 20,
      "passCount": 18,
      "passRate": 90.0
    },
    {
      "coachName": "李教练",
      "rating": 4.5,
      "coachYears": 10,
      "studentCount": 8,
      "examCount": 12,
      "passCount": 9,
      "passRate": 75.0
    }
  ]
}
```

**说明：**
- `series` 数据可直接渲染柱状图（通过率排名）
- `detailData` 包含每项明细，可用于鼠标悬浮 tooltip 或下方表格展示

### 4. 收入看板

```
GET /statistics/revenue-summary
```

> 月度收入趋势 + 当月收入来源分布 + 收支汇总。

**响应 data：**
```json
{
  "monthlyChart": {
    "title": { "text": "近12个月收入趋势" },
    "xAxis": { "type": "category", "data": ["2025-07", "2025-08", "...", "2026-06"] },
    "yAxis": { "type": "value" },
    "series": [
      { "name": "收入", "type": "bar", "data": [0, 3980, 200, 0, ...] }
    ]
  },
  "bizTypeChart": {
    "title": { "text": "当月收入来源", "left": "center" },
    "series": [
      {
        "name": "收入来源",
        "type": "pie",
        "radius": "50%",
        "data": [
          { "name": "报名费", "value": 3980.00 },
          { "name": "考试费", "value": 200.00 },
          { "name": "合场费", "value": 200.00 }
        ]
      }
    ]
  },
  "summary": {
    "total_paid": 4380.00,
    "paid_count": 3,
    "total_pending": 2000.00,
    "pending_count": 2,
    "total_refunded": 0,
    "refunded_count": 0
  }
}
```

**说明：**
- `monthlyChart`：近12月收入柱状图，直接传入 ECharts option
- `bizTypeChart`：当月各类收入占比饼图，直接传入 ECharts option
- `summary`：汇总数字，可用于顶部统计卡片展示
