# 统计报表

基础路径: `/statistics` — 返回 ECharts 兼容的 JSON 数据，前端可直接用于图表渲染。

| 方法 | URL | 图表类型 | 说明 | 需要登录 |
|------|-----|----------|------|----------|
| GET | `/statistics/registration-trend` | 折线图 | 近30天每日报名人数 | 是 |
| GET | `/statistics/pass-rate` | 饼图 | 考试合格/不合格分布 | 是 |
| GET | `/statistics/coach-workload` | 柱状图 | 各教练的约课数量 | 是 |

## GET /statistics/registration-trend

近30天每日学员报名人数折线图，无数据的日期补 0。

```bash
curl http://localhost:9500/statistics/registration-trend \
  -H "Authorization: Bearer <token>"
```

```json
{
  "state": 20000,
  "message": null,
  "data": {
    "title": { "text": "近30天报名趋势" },
    "xAxis": { "type": "category", "data": ["2026-04-22", "2026-04-23", "..."] },
    "yAxis": { "type": "value", "minInterval": 1 },
    "series": [{ "name": "报名人数", "type": "line", "smooth": true, "data": [0, 1, 3, "..."] }]
  }
}
```

## GET /statistics/pass-rate

所有科目考试合格与不合格人数的饼图分布。

```bash
curl http://localhost:9500/statistics/pass-rate \
  -H "Authorization: Bearer <token>"
```

```json
{
  "state": 20000,
  "message": null,
  "data": {
    "title": { "text": "考试合格率", "left": "center" },
    "tooltip": { "trigger": "item", "formatter": "{b}: {c} ({d}%)" },
    "legend": { "orient": "vertical", "left": "left" },
    "series": [{
      "name": "考试结果", "type": "pie", "radius": "50%",
      "data": [
        { "name": "合格", "value": 85 },
        { "name": "不合格", "value": 15 }
      ]
    }]
  }
}
```

> 当没有任何考试记录时，`data.series[0].data` 返回 `[{ "name": "暂无数据", "value": 1 }]`。

## GET /statistics/coach-workload

各教练已完成和已确认的约课数量柱状图，按约课数降序排列。

```bash
curl http://localhost:9500/statistics/coach-workload \
  -H "Authorization: Bearer <token>"
```

```json
{
  "state": 20000,
  "message": null,
  "data": {
    "title": { "text": "教练工作量统计" },
    "xAxis": { "type": "category", "data": ["张教练", "李教练"] },
    "yAxis": { "type": "value", "minInterval": 1 },
    "series": [{ "name": "约课数", "type": "bar", "data": [5, 3] }]
  }
}
```
