# 系统配置管理 & 报表导出 — 前端接口文档

> 新增功能：系统配置管理页面、统计报表 Excel 导出、特种车辆进度适配。

---

## 一、系统配置管理（管理员专用）

**基础路径**: `/configs`

> 仅管理员可调用。`config` 表以 `configKey`（字符串）为自然主键，无自增 ID。

### 1.1 配置项数据结构

```json
{
  "configKey": "school_name",
  "configValue": "阳光驾校",
  "description": "驾校名称，用于 PDF 生成",
  "createTime": "2026-06-10T12:00:00",
  "updateTime": "2026-06-10T12:00:00"
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `configKey` | String | 配置键（主键，字符串） |
| `configValue` | String | 配置值 |
| `description` | String | 说明 |
| `createTime` | String | 创建时间 |
| `updateTime` | String | 修改时间 |

### 1.2 接口列表

| 方法 | 路径 | 用途 | 说明 |
|------|------|------|------|
| `GET` | `/configs` | 查询所有配置项 | 返回完整 `Config` 列表，不含分页 |
| `GET` | `/configs/{configKey}` | 按 Key 查询 | 如 `/configs/school_name` |
| `POST` | `/configs` | 新增配置项 | Body 传完整实体（`configKey` + `configValue` + 可选 `description`） |
| `PUT` | `/configs/{configKey}` | 修改配置项 | Body 传需修改的字段（`configValue` / `description`） |
| `DELETE` | `/configs/{configKey}` | 删除配置项 | 物理删除，不可恢复 |

### 1.3 配置项参考

以下为系统用到的配置项，**请勿在管理页面删除**：

| configKey | 默认值 | 用途 |
|-----------|--------|------|
| `school_name` | 阳光驾校 | 驾校名称，用于 PDF 报表和文件下载 |
| `exam_pass_score` | 90 | 各科目合格分数线 |
| `retake_training_fee` | 300 | 二次培训（补考培训）默认费用 |

> 管理员可在配置管理页面**新增其他配置项**，上述三项为代码中硬编码引用的系统级配置。

## 二、统计报表 Excel 导出（管理员专用）

**基础路径**: `/statistics`

> 四个统计报表对应的原始 JSON 接口（`GET`）已在之前版本提供，本次新增 **Excel 导出** 功能，均返回 `.xlsx` 文件流。

### 2.1 报名趋势导出

```
POST /statistics/registration-trend/export-excel
```

> 返回 Excel（两列：日期、报名人数），内容与 `GET /statistics/registration-trend` 相同。

### 2.2 各科通过率导出

```
POST /statistics/pass-rate/export-excel
```

> 返回 Excel（首列月份 + 各科目分别一列），内容与 `GET /statistics/pass-rate` 相同。

### 2.3 教练效能排名导出

```
POST /statistics/coach-workload/export-excel
```

> 返回 Excel（7 列：教练姓名 / 通过率(%) / 综合评分 / 执教年限 / 带教学员数 / 考试人次 / 通过人次），内容与 `GET /statistics/coach-workload` 的 `detailData` 相同。

### 2.4 收入看板导出

```
POST /statistics/revenue-summary/export-excel
```

> 返回一个包含 **3 个 Sheet** 的工作簿：
> - **Sheet 1 - 月度收入趋势**：月份 + 收入(元)，对应 JSON 的 `monthlyChart`
> - **Sheet 2 - 收入来源分布**：来源 + 金额(元)，对应 JSON 的 `bizTypeChart`
> - **Sheet 3 - 收支汇总**：指标 + 金额(元)，对应 JSON 的 `summary`

### 导出接口用法

**请求方式**：`POST`，无请求体

**前端对接**：使用 `window.open` 或 `<a>` 直接访问接口，或用 axios 配置 `responseType: 'blob'` 下载：

```javascript
// 方式一：直接跳转
window.open('/api/statistics/registration-trend/export-excel', '_blank')

// 方式二：axios 下载
axios.post('/statistics/registration-trend/export-excel', null, {
  responseType: 'blob',
  headers: { Authorization: 'Bearer ' + token }
}).then(res => {
  const url = URL.createObjectURL(new Blob([res.data]))
  const a = document.createElement('a')
  a.href = url
  a.download = '报名趋势.xlsx'
  a.click()
})
```

## 三、特种车辆进度适配

### 3.1 进度查询接口

```
GET /students/{studentId}/progress
```

> 接口路径不变，返回新增以下字段：

```json
{
  "studentId": 7,
  "realName": "测试学员N1",
  "licenseType": "N1",
  "examMode": 2,
  "certName": "N1驾驶证",
  "subjects": [
    {
      "subject": 1,
      "examMode": 2,
      "description": "科目一（特种车）",
      "requiredHours": 0,
      "trainedHours": 0,
      "status": "passed",
      "score": 95,
      "remainingHours": 0,
      "examItems": []
    },
    {
      "subject": 3,
      "examMode": 2,
      "description": "科目三（特种车）",
      "requiredHours": 30,
      "trainedHours": 25,
      "status": "learning",
      "remainingHours": 5,
      "examItems": []
    }
  ],
  "progressPercent": 50,
  "allPassed": false,
  "certificate": null
}
```

### 3.2 新增字段说明

| 字段 | 类型 | 所属层级 | 说明 |
|------|------|---------|------|
| `examMode` | int | 根节点 | 1=小汽车(4科), 2=特种车辆(2科) |
| `certName` | String | 根节点 | 证书名称，如 "N1驾驶证" |
| `allPassed` | boolean | 根节点 | 是否全部科目已通过 |
| `certificate` | String | 根节点 | 仅 `examMode=2 && allPassed=true` 时有值，表示可结业领证 |
| `examMode` | int | subjects[] | 每个科目的考试模式（同一车型所有科目共享） |
| `examItems` | String[] | subjects[] | 考试项目列表（逗号分隔转数组），如科目二可能返回 `["倒车入库","侧方停车"]` |

### 3.3 前端展示建议

- **examMode = 1（小汽车）**：显示 4 个科目（科目一→科目四），与原来一致
- **examMode = 2（特种车辆）**：显示 2 个科目（科目一、科目三），**跳过科目二和科目四**
- 当 `allPassed = true` 且 `certificate` 不为 null 时，在进度页面底部显示 **"可结业领证"** 提示，文案可展示证书名称

## 四、前端调整清单

1. **配置管理页（管理员）**：新增页面，调用 `/configs` 系列接口，提供配置键值对的增删改查
2. **统计报表导出按钮**：四个报表页面各新增一个"导出 Excel"按钮，调用对应的 `POST /statistics/*/export-excel` 接口
3. **学习进度页**：
   - 根据 `examMode` 判断显示 4 科或 2 科布局
   - 科目列表展示 `examItems`（考试项目）
   - 全部通过且显示 `certificate` 时展示"可结业领证"标识
