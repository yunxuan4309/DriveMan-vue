# 文件管理接口文档

## 概述

统一的文件管理体系，支持学员/教练/管理员三端的文件上传、查询、预览、下载功能。
文件通过 `bizType` 分类归档，通过 `userId` 控制归属权限。

---

## 通用说明

### 认证方式

所有接口（除 `/uploads/**` 静态资源外）需要在请求头中携带 Token：

```
Authorization: Bearer <token>
```

### 角色说明

| 值 | 角色 |
|----|------|
| 1 | 学员 |
| 2 | 教练 |
| 3 | 管理员 |

### 统一响应

```json
{
  "state": 200,
  "message": null,
  "data": { ... }
}
```

## 1. 上传文件

```
POST /files/upload
Content-Type: multipart/form-data
```

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `userId` | Integer | ✅ | 文件归属人 ID |
| `file` | MultipartFile | ✅ | 上传的文件（单文件最大 5MB） |
| `fileType` | String | ✅ | 文件分类，见下方 |
| `bizType` | String | ❌ | 业务类型，见下方 |
| `bizId` | Integer | ❌ | 业务记录 ID |

### fileType（文件分类）

| 值 | 说明 | 允许格式 |
|----|------|---------|
| `id_card_front` | 身份证正面 | jpg/jpeg/png/bmp/webp |
| `id_card_back` | 身份证反面 | jpg/jpeg/png/bmp/webp |
| `physical_exam` | 体检表 | jpg/jpeg/png/bmp/webp/pdf |
| `registration_pdf` | 报名表 PDF | pdf |
| `admission_ticket` | 准考证 PDF | pdf |
| `training_record` | 培训记录表 PDF | pdf |
| `coach_qualification` | 教练资质材料 | jpg/jpeg/png/bmp/webp/pdf |

### bizType（业务类型）

| 值 | 对应业务 | 使用者 |
|----|---------|--------|
| `user_profile` | 用户资料（身份证、头像等） | 学员/教练/管理员 |
| `enrollment` | 学员报名材料 | 学员 |
| `exam_ticket` | 准考证 | 学员（系统生成） |
| `registration_form` | 报名表 | 学员（系统生成） |
| `training_record` | 培训记录表 | 学员（系统生成） |
| `physical_exam` | 体检报告 | 学员 |
| `license_upgrade` | 增驾/准教车型变更材料 | 学员/教练 |
| `coach_qualification` | 教练资质材料 | 教练 |

### 权限

- 学员/教练：**只能上传自己**的文件（`userId` 必须等于当前登录用户 ID）
- 管理员：可上传任意用户的文件

### 响应

```json
{
  "state": 200,
  "data": {
    "id": 1,
    "userId": 5,
    "fileName": "身份证正面.jpg",
    "filePath": "user_profile/5_20260603143000123_身份证正面.jpg",
    "fileSize": 102400,
    "mimeType": "image/jpeg",
    "fileType": "id_card_front",
    "bizType": "user_profile",
    "bizId": 5,
    "uploadTime": "2026-06-03T14:30:00",
    "createTime": "2026-06-03T14:30:00",
    "updateTime": "2026-06-03T14:30:00",
    "isDeleted": 0
  }
}
```

## 2. 查询当前用户自己的文件

```
GET /files/my[?bizType=][&fileType=][&keyword=]
```

查询当前登录用户的文件列表，支持多维度过滤。

### 请求参数（均可选）

| 参数 | 类型 | 说明 |
|------|------|------|
| `bizType` | String | 按业务类型精确过滤 |
| `fileType` | String | 按文件分类精确过滤 |
| `keyword` | String | 按文件名模糊搜索 |

### 权限

- 学员/教练/管理员：查自己的文件

### 常用查询场景

```
# 找准考证（学员）
GET /files/my?bizType=exam_ticket

# 找培训记录表（学员）
GET /files/my?bizType=training_record

# 找报名表（学员）
GET /files/my?bizType=registration_form

# 找自己的资料图片（学员/教练）
GET /files/my?bizType=user_profile

# 找资质材料（教练）
GET /files/my?bizType=coach_qualification

# 按文件名搜索
GET /files/my?keyword=王小明

# 组合条件
GET /files/my?bizType=exam_ticket&keyword=科目三
```

### 响应

```json
{
  "state": 200,
  "data": [
    {
      "id": 1,
      "userId": 5,
      "fileName": "准考证_王小明_科目三.pdf",
      "filePath": "admission_ticket/5_ticket_session_3_2026-06-03.pdf",
      "fileSize": 204800,
      "mimeType": "application/pdf",
      "fileType": "admission_ticket",
      "bizType": "exam_ticket",
      "bizId": 15,
      "uploadTime": "2026-06-03T10:00:00"
    }
  ]
}
```

## 3. 按业务查询附件

```
GET /files/biz/{bizType}/{bizId}
```

查询某条业务记录关联的所有文件。

### 参数

| 参数 | 说明 |
|------|------|
| `bizType` | 业务类型 |
| `bizId` | 业务记录 ID |

### 权限

- 普通用户：只能查到文件归属人是自己的记录
- 管理员：查询所有

### 示例

```
# 查考试报名记录 ID=15 的准考证
GET /files/biz/exam_ticket/15
```

---

## 4. 查询文件详情

```
GET /files/{id}
```

按文件 ID 查询单条记录。

### 权限

- 文件归属人可查
- 管理员可查

## 5. 管理员按条件查询文件

```
GET /files/admin/query?realName=&role=
```

管理员按用户真实姓名和角色来查找文件，不需要知道用户 ID。

### 请求参数（至少传一个）

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `realName` | String | ❌ | 用户真实姓名（模糊搜索） |
| `role` | Integer | ❌ | 角色: 1-学员, 2-教练 |

### 权限

- **仅管理员**可用

### 示例

```
# 查某个学员的所有文件
GET /files/admin/query?realName=王小明&role=1

# 查所有学员的文件
GET /files/admin/query?role=1

# 按姓名搜（不限角色）
GET /files/admin/query?realName=王

# 查所有教练的文件
GET /files/admin/query?role=2

# 查姓张的教练的文件
GET /files/admin/query?realName=张&role=2
```

### 响应

```json
{
  "state": 200,
  "data": [
    {
      "id": 1,
      "userId": 5,
      "realName": "王小明",
      "fileName": "准考证_王小明_科目三.pdf",
      "filePath": "admission_ticket/5_ticket_session_3_2026-06-03.pdf",
      "fileSize": 204800,
      "mimeType": "application/pdf",
      "fileType": "admission_ticket",
      "bizType": "exam_ticket",
      "bizId": 15,
      "uploadTime": "2026-06-03T10:00:00",
      "createTime": "2026-06-03T10:00:00"
    }
  ]
}
```

> 每条记录附带 `realName` 字段，方便管理端表格直接展示用户姓名。

## 6. 下载/预览文件

```
GET /files/{id}/download?preview=false
```

### 参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
|------|------|------|------|------|
| `preview` | boolean | ❌ | `false` | `false`=下载, `true`=预览 |

### 权限

- 文件归属人可操作
- 管理员可操作

### 前端调用方式

```html
<!-- 下载 -->
<a href="/files/{id}/download" download>下载</a>

<!-- 预览（新标签页打开 PDF） -->
<a href="/files/{id}/download?preview=true" target="_blank">预览</a>

<!-- iframe 嵌入预览 -->
<iframe src="/files/{id}/download?preview=true"></iframe>

<!-- 图片直接显示（通过静态资源 URL，无需 Token） -->
<img src="/uploads/user_profile/5_xxx.jpg">
```

### 响应头

| 模式 | Content-Type | Content-Disposition |
|------|-------------|-------------------|
| 下载 | `application/octet-stream` | `attachment; filename*=UTF-8''xxx.pdf` |
| 预览 | 文件实际类型（如 `application/pdf`） | `inline; filename*=UTF-8''xxx.pdf` |

## 7. 生成培训记录表 PDF

```
POST /files/generate-training-record?studentId=5
```

学员学时达标后调用，系统生成培训记录表 PDF 并自动入库。

### 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `studentId` | Integer | ✅ | 学员 ID |

### 权限

- 学员：只能为自己生成
- 管理员：可为任意学员生成

### 生成内容

学员姓名、身份证号、教练姓名、驾校名称、各科目累计学时、生成日期

### 响应

```json
{
  "state": 200,
  "data": {
    "id": 10,
    "userId": 5,
    "fileName": "培训记录表_王小明.pdf",
    "filePath": "training_record/5_record_2026-06-03.pdf",
    "bizType": "training_record",
    "bizId": 5
  }
}
```

生成后通过 `GET /files/my?bizType=training_record` 查到，再预览或下载。

---

## 8. 删除文件（管理员专用）

```
DELETE /files/{id}
```

逻辑删除（`is_deleted = 1`），磁盘文件保留。

## 三端功能速查

### 学员端

| 功能 | 接口 |
|------|------|
| 上传身份证等资料 | `POST /files/upload` + `bizType=user_profile` |
| 上传报名材料 | `POST /files/upload` + `bizType=enrollment` |
| 查看自己的所有文件 | `GET /files/my` |
| 找准考证 | `GET /files/my?bizType=exam_ticket` |
| 找报名表 | `GET /files/my?bizType=registration_form` |
| 找培训记录表 | `GET /files/my?bizType=training_record` |
| 生成培训记录表 | `POST /files/generate-training-record` |
| 预览 PDF | `GET /files/{id}/download?preview=true` |
| 下载文件 | `GET /files/{id}/download` |

### 教练端

| 功能 | 接口 |
|------|------|
| 上传头像/证件 | `POST /files/upload` + `bizType=user_profile` |
| 上传资质材料 | `POST /files/upload` + `bizType=coach_qualification` |
| 查看自己的文件 | `GET /files/my` |
| 预览/下载 | `GET /files/{id}/download?preview=true` |

> 教练不可查看学员的文件。

### 管理员端

| 功能 | 接口 |
|------|------|
| 查任意用户的文件 | `GET /files/admin/query?realName=&role=` |
| 按业务查文件 | `GET /files/biz/{bizType}/{bizId}` |
| 替用户上传文件 | `POST /files/upload` + 任意 `userId` |
| 删除文件 | `DELETE /files/{id}` |
| 预览/下载任意文件 | `GET /files/{id}/download?preview=` |

---

## 注意事项

1. **Token** — 除 `/uploads/**` 静态资源外，所有接口均需 `Authorization: Bearer <token>`
2. **文件大小** — 单文件最大 5MB，单次请求最大 20MB
3. **静态资源** — `GET /uploads/{filePath}` 直接访问文件（无需 Token），适用于 `<img>` 展示图片
4. **预览** — PDF 和图片类型支持浏览器预览，前端用 `<a target="_blank">` 或 `<iframe>`
