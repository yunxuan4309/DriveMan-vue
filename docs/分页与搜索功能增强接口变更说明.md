# 功能增强 — 分页与搜索接口变更说明

> 日期：2026-06-10

涉及 4 个模块：考试审核、教练申请审核、教练分配、文件管理。均为后端新增分页 + 条件搜索能力，前端需要对接。

---

## ⑧ 考试审核 — 新增学员姓名搜索

### 接口

```http
GET /exam-registrations?page=1&size=10&status=0&keyword=张三
```

### 新增参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| keyword | String | 否 | 学员真实姓名模糊搜索 |

### 示例

**请求：** 查看待审核记录，搜索学员姓名含"张"
```http
GET /exam-registrations?page=1&size=20&status=0&keyword=张
```

**响应：** 与现有 `GET /exam-registrations` 一致，返回 `Page<Map<String, Object>>`

---

## ⑨ 教练申请审核 — 新增分页查询接口

### 接口（新建）

```http
GET /coach-applications?page=1&size=10&status=0&keyword=张三
```

原有的 `GET /coach-applications/pending` 保持不变，仅用于查全部待审核。

### 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | 否 | 页码，默认 1 |
| size | int | 否 | 每页条数，默认 10 |
| status | int | 否 | 状态：0-待审核, 1-通过, 2-拒绝 |
| keyword | String | 否 | 学员姓名模糊搜索 |

### 响应字段

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 申请记录 ID |
| studentId | int | 学员 ID |
| studentName | String | 学员姓名 |
| coachId | int | 目标教练 ID |
| coachName | String | 目标教练姓名 |
| sourceCoachId | int | 发起移交的教练 ID（NULL 表示学员自主申请） |
| sourceCoachName | String | 发起移交的教练姓名 |
| transferReason | String | 教练移交原因 |
| applyType | String | 申请类型：学员申请 / 教练移交 |
| status | int | 0-待审核, 1-通过, 2-拒绝 |
| applyTime | LocalDateTime | 申请时间 |
| auditTime | LocalDateTime | 审核时间 |
| auditReason | String | 拒绝原因 |

### 示例

**请求：** 查看所有待审核记录（第 1 页，每页 10 条）
```http
GET /coach-applications?page=1&size=10&status=0
```

**请求：** 搜索学员姓名含"王"的申请记录
```http
GET /coach-applications?page=1&size=10&keyword=王
```

**响应：**
```json
{
  "state": 200,
  "data": {
    "records": [
      {
        "id": 1,
        "studentId": 5,
        "studentName": "王小明",
        "coachId": 2,
        "coachName": "李教练",
        "sourceCoachId": null,
        "sourceCoachName": null,
        "applyType": "学员申请",
        "status": 0,
        "applyTime": "2026-06-10T10:00:00",
        "auditTime": null,
        "auditReason": null
      }
    ],
    "total": 1,
    "size": 10,
    "current": 1
  }
}
```

## ⑩ 教练分配 — 新增分页 + 学员/教练姓名搜索

### 接口（改造）

```http
GET /coach-assignments?page=1&size=10&studentName=王&coachName=李
```

### 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | 否 | 页码，默认 1 |
| size | int | 否 | 每页条数，默认 10 |
| studentName | String | 否 | 学员姓名模糊搜索 |
| coachName | String | 否 | 教练姓名模糊搜索 |

### 响应

返回 `Page<Map<String, Object>>`，每条记录包含：

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 绑定记录 ID |
| studentId | int | 学员 ID |
| studentName | String | 学员姓名 |
| coachId | int | 教练 ID |
| coachName | String | 教练姓名 |
| bindTime | LocalDateTime | 绑定时间 |

### 前端适配说明

**返回值类型变更：** `List` → `Page`，需要按分页组件对接。

### 示例

**请求：** 查第 1 页，每页 20 条
```http
GET /coach-assignments?page=1&size=20
```

**请求：** 按学员姓名搜索
```http
GET /coach-assignments?page=1&size=10&studentName=王
```

## ⑪ 文件管理 — 两处接口增加分页

### ⑪-a 我的文件

```http
GET /files/my?page=1&size=10&bizType=enrollment&keyword=报名
```

#### 新增参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | 否 | 页码，默认 1 |
| size | int | 否 | 每页条数，默认 10 |

原有参数（bizType / fileType / keyword）不变。

**返回值类型变更：** `List<File>` → `Page<File>`

---

### ⑪-b 管理员查询文件

```http
GET /files/admin/query?page=1&size=10&realName=张&role=1
```

#### 参数变更

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | 否 | 页码，默认 1（**新增**） |
| size | int | 否 | 每页条数，默认 10（**新增**） |
| realName | String | 否 | 用户真实姓名模糊搜索 |
| role | int | 否 | 角色：1-学员, 2-教练 |

> **注意：** `realName` 和 `role` 不再强制要求至少传一个（不传则查所有用户的文件）。

#### 响应

返回 `Page<Map<String, Object>>`，每条记录包含：

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 文件 ID |
| userId | int | 上传者 ID |
| realName | String | 上传者真实姓名 |
| fileName | String | 文件名 |
| filePath | String | 文件路径 |
| fileSize | Long | 文件大小（字节） |
| mimeType | String | MIME 类型 |
| fileType | String | 文件分类 |
| bizType | String | 业务类型 |
| bizId | int | 业务记录 ID |
| uploadTime | LocalDateTime | 上传时间 |
| createTime | LocalDateTime | 创建时间 |

**返回值类型变更：** `List<Map<String, Object>>` → `Page<Map<String, Object>>`

---

## 汇总对照

| 模块 | 接口 | 变更内容 |
|------|------|----------|
| ⑧ 考试审核 | `GET /exam-registrations` | 新增 `keyword` 参数（学员姓名搜索） |
| ⑨ 教练申请审核 | `GET /coach-applications` | **新增接口**，支持分页 + status + keyword |
| ⑩ 教练分配 | `GET /coach-assignments` | 改为分页返回，新增 `studentName` / `coachName` 搜索 |
| ⑪-a 我的文件 | `GET /files/my` | 改为分页返回，新增 `page` / `size` 参数 |
| ⑪-b 管理员查文件 | `GET /files/admin/query` | 改为分页返回，新增 `page` / `size` 参数 |

其中 ⑩、⑪-a、⑪-b **返回值从 `List` 变更为 `Page`**，前端需要对应调整分页组件。
